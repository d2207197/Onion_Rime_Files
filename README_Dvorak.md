# 洋蔥注音 Plus Dvorak 版 使用指南

本方案基於[洋蔥注音 Plus 版](https://github.com/oniondelta/Onion_Rime_Files)，
針對 **Dvorak 鍵盤 layout** 調整，讓使用者用 Dvorak 的物理鍵位輸入大千式注音。

## 安裝

將以下檔案放入 Rime 使用者目錄（macOS: `~/Library/Rime/`）：

- `bopomo_onionplus_dvorak.schema.yaml` — Dvorak 版方案定義
- `element_bopomo_dvorak.yaml` — Dvorak 版按鍵設定
- 其餘所有檔案與原版洋蔥注音 Plus 共用（字典、lua、opencc 等）

在 `default.custom.yaml` 中加入：

```yaml
patch:
  schema_list:
    - schema: bopomo_onionplus_dvorak
```

重新部署鼠鬚管即可使用。

---

## 設計原則

**物理鍵位不變**：所有 `Shift+字母` 快捷鍵維持與 QWERTY 原版相同的物理位置。
QWERTY 使用者切換到 Dvorak 版，手指位置和原版完全一致，不需重新記憶。

## 選字操作

候選字選單出現後，共 8 個候選字，可用左手或右手選字：

### 右手選字（主要）

| 候選 | 按鍵 | 物理位置 (QWERTY) |
|:----:|:----:|:-----------------:|
| 1 | `Shift+F` | Y |
| 2 | `Shift+D` | H |
| 3 | `Shift+B` | N |
| 4 | `Shift+G` | U |
| 5 | `Shift+H` | J |
| 6 | `Shift+M` | M |
| 7 | `Shift+C` | I |
| 8 | `Shift+T` | K |

### 左手選字（輔助）

| 候選 | 按鍵 | 物理位置 (QWERTY) |
|:----:|:----:|:-----------------:|
| 2 | `Shift+A` | A |
| 4 | `Shift+O` | S |
| 5 | `Shift+Q` | X |
| 7 | `Shift+E` | D |
| 8 | `Shift+J` | C |

> 候選 1、3、6 的左手物理位置在 Dvorak 上為標點鍵（`'` `,` `.`），
> Shift 後產生 `"` `<` `>`，與注音輸入衝突，因此停用。

### 翻頁

| 操作 | 按鍵 |
|------|------|
| 上一頁 | `Shift+空白` 以外的：`↑`（選字時）、`←`（選字時） |
| 下一頁 | `Shift+空白`、`→`（選字時） |

## 快捷鍵

所有快捷鍵維持與 QWERTY 原版相同的**物理位置**：

### 編輯操作

| 功能 | 按鍵 | 物理位置 (QWERTY) | 說明 |
|------|------|:-----------------:|------|
| 補一聲 / 空格 | `Shift+N` | L | 選字時補一聲，或輸出空格 |
| 上屏原始輸入 | `Shift+R` | O | 直接上屏鍵盤原始字元 |
| 上屏注音文 | `Shift+U` | F | 上屏 preedit 注音文字 |
| 上屏 comment | `Shift+P` | R | 上屏提示碼內容 |
| 退格 | `Shift+L` | P | 刪除前一字 |
| 刪除 | `Shift+Y` | T | 刪除後一字 |
| 取消 | `Shift+I` | G | Escape，清除輸入 |
| 移到開頭 | `Shift+K` | V | Home |
| 移到結尾 | `Shift+X` | B | End |

### 切分移位

| 功能 | 按鍵 |
|------|------|
| 左移一個注音 | `↑`、`Shift+←` |
| 右移一個注音 | `Shift+→` |
| 右移一整個字 | `→` |
| 回到輸入列 | `Tab`（選字狀態時） |

### 中英文與模式切換

| 功能 | 按鍵 |
|------|------|
| 切換中/英文 | `Control+Shift+2` 或 `Shift+空白`（非選字時） |
| 切換方案 | `Control+~` 或 `F4` |
| 切換全/半形 | `Control+Shift+3` |
| 切換繁/簡 | `Control+Shift+5` |
| 切換標點模式 | `Control+Shift+4` |

## 外掛語言與特殊功能

洋蔥 Plus 版內建多種外掛語言輸入，在注音模式下輸入特定前綴啟動：

| 功能 | 前綴 | 提示 | 說明 |
|------|:----:|------|------|
| 英文 | `3` | 《Easy》 | 英漢字典查詞 |
| 拉丁字母 | `4` | 《拉》 | 含重音符號的拉丁字母 |
| 日文 | `6` | 《日》 | 平假名、片假名、漢字 |
| 俄文 | `66` | 《俄》 | 西里爾字母 |
| 韓文 | `7` | 《韓》 | 韓語 HNC |
| 希臘文 | `77` | 《希》 | 希臘字母 |
| 全形 | `][` | 《全》 | 全形英數字 |
| 注音文 | `';` | 《注音文》 | 輸出注音符號本身 |
| emoji | `` ` `` | 《emoji集》 | 表情符號系列 |
| 倉頡反查 | `=` | 《查》 | 用倉頡碼反查注音 |
| 數學/功能 | `'/` | — | 日期時間、計算機等 lua 功能 |

> **注意**：上面的前綴是 Dvorak 鍵盤上的字元。
> 例如按物理 `3` 鍵會輸入 `3`（數字列不變），啟動英文模式。

### 英文模式範例

1. 按 `3` 進入英文模式（提示《Easy》）
2. 輸入英文單字
3. 選字上屏

### emoji 模式範例

1. 按 `` ` ``（ESC 下方的鍵）進入 emoji 模式
2. 輸入注音拼出分類名稱（如「笑」「愛」「食」）
3. 選擇想要的 emoji

## 標點符號

標點符號由 `punct_bopomo` 處理，以下是常用標點的輸入方式：

### 直接輸入（半形模式）

| 按鍵 | 輸出 | 說明 |
|:----:|:----:|------|
| `<` (Shift+,) | ， | 逗號 |
| `>` (Shift+.) | 。 | 句號 |
| `?` (Shift+/) | ？ | 問號 |
| `:` (Shift+;) | ： | 冒號 |
| `"` (Shift+') | ； | 分號 |
| `~` | ～ | 波浪號 |
| `!` | ！ | 驚嘆號 |
| `(` | （ | 左括號 |
| `)` | ） | 右括號 |
| `{` | 〔 | 左方括號 |
| `}` | 〕 | 右方括號 |

### 多選標點（按鍵後從候選選擇）

| 按鍵 | 首選 | 物理位置 (QWERTY) | 說明 |
|:----:|:----:|:-----------------:|------|
| `-` | 、 | ' (Enter 旁) | **頓號**系列（含—‧·等）。Dvorak 專屬覆蓋，物理位置與 QWERTY 原版相同 |
| `[` | 「 | - | 左引號系列（含〈〔'等） |
| `]` | 」 | = | 右引號系列（含〉〕'等） |
| `=` | ＝ | ] | 等號系列（含々〃等） |
| `\` | ＼ | | 反斜線系列（composing 時作為音節分隔符） |
| `\|` | ｜ | | 豎線系列 |

> **備註**：`'` 鍵（物理 Q）在 punct_bopomo 中也對應頓號，
> 但因為它同時是注音 ㄆ 的輸入鍵，僅在非輸入狀態下才能觸發。
> 建議使用 `-` 鍵作為主要頓號輸入方式。

## 與 QWERTY 原版的差異摘要

| 項目 | QWERTY 原版 | Dvorak 版 |
|------|-------------|-----------|
| 注音鍵位 | 大千式 QWERTY | 大千式 Dvorak（同物理位置） |
| 選字鍵數 | 8 個（左右手各 8） | 8 個右手 + 5 個左手 |
| 選字標籤 | 𝟷𝚀𝚈 等 (Unicode 等寬) | 1F 2D 3B 等 (ASCII) |
| 快捷鍵 | Shift+QWERTY字母 | Shift+Dvorak字母（同物理位置） |
| 字典/prism | bopomo_onionplus | 字典共用，prism 獨立編譯 |
| 其他設定 | element_bopomo | element_bopomo_dvorak |
