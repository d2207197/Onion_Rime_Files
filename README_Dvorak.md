# 洋蔥注音 Plus Dvorak 版 使用指南

本方案基於[洋蔥注音 Plus 版](https://github.com/oniondelta/Onion_Rime_Files)，
針對 **Dvorak 鍵盤 layout** 調整，讓使用者用 Dvorak 的物理鍵位輸入大千式注音。

## 安裝

將以下檔案放入 Rime 使用者目錄（macOS: `~/Library/Rime/`）：

- `bopomo_onionplus_dvorak.schema.yaml` — Dvorak 版方案定義
- `element_bopomo_dvorak.yaml` — Dvorak 版按鍵設定
- 其餘所有檔案與原版洋蔥注音 Plus 共用（字典、lua、opencc 等）

在 `default.custom.yaml` 中加入：

```yaml
patch:
  schema_list:
    - schema: bopomo_onionplus_dvorak
```

重新部署鼠鬚管即可使用。

---

## 設計原則

1. **右手兩排選字**：HTNS / MWVZ 八鍵選字，上排 GCR 空出給功能鍵
2. **Emacs 風格游標**：F=forward、B=backward、D=done，圍繞九宮格
3. **助記鍵**：所有功能鍵都有助記（vi/Emacs/英文縮寫），不需死記
4. **一個功能一個鍵**：不重複綁定，簡潔明確

## 選字操作

候選字選單出現後，共 8 個候選字，使用右手中排+下排選字：

```
┌──────────┬──────────┬──────────┬──────────┐
│ H = 候選1│ T = 候選2│ N = 候選3│ S = 候選4│
├──────────┼──────────┼──────────┼──────────┤
│ M = 候選5│ W = 候選6│ V = 候選7│ Z = 候選8│
└──────────┴──────────┴──────────┴──────────┘
```

- **Enter** = 確認第 1 候選（平時使用）
- **Shift+H** = 選第 1 候選（回頭重選字時方便）
- **Tab** = 離開選字模式，回到編輯狀態
- 繼續輸入注音也會自動離開選字模式

> 空閒鍵（保留給未來功能）：右手 G, C, R, D, S, L ／左手 Y, U, I, X

### 翻頁（vi 風格）

| 操作 | 按鍵 | 助記 |
|------|:----:|------|
| 上一頁 | `Shift+K` | vi k=up |
| 下一頁 | `Shift+J` | vi j=down |
| 上一頁 | `←` | 選字狀態時 |
| 下一頁 | `→` | 選字狀態時 |

## 快捷鍵

### Shift+鍵 功能總覽

```
Dvorak 右手：                              Dvorak 左手：
  F(fwd)  G(-)  C(-)  R(-)                  P(注音文) ...
  D(---) [H=1  T=2  N=3  S=4]               A(ahead) O(orig) E(end)
  B(bwd) [M=5  W=6  V=7  Z=8]               Q(quit)  J(pgdn) K(pgup)
```

### 右手 — Emacs 游標與操作（composing 時）

| 按鍵 | 功能 | 助記 |
|------|------|------|
| `Shift+F` | 右移一個字 | Emacs C-**f** = **f**orward |
| `Shift+B` | 左移回退一個字 | Emacs C-**b** = **b**ackward |

### 左手 — 導航

| 按鍵 | 功能 | 助記 |
|------|------|------|
| `Shift+A` | Home（跳到開頭） | **A** = **a**head / Emacs C-a |
| `Shift+E` | End（跳到結尾） | Emacs C-**e** |
| `Shift+J` | Page Down | vi **j**=down |
| `Shift+K` | Page Up | vi **k**=up |

### 左手 — 操作與上屏

| 按鍵 | 功能 | 助記 |
|------|------|------|
| `Shift+Q` | Escape（清除輸入） | **Q**=Quit |
| `Shift+O` | 上屏原始輸入 | **O**=Original |
| `Shift+P` | 上屏注音文 | **P**=Phonetic |

### 刪除操作

| 按鍵 | 效果 | 說明 |
|------|------|------|
| `BackSpace` | 刪除整組注音（一個字） | Rime 共用設定 |
| `Shift+BackSpace` | 刪除一個注音字符 | 精細刪除 |
| `\` | 刪除一個注音字符 | 選字狀態時可用 |
| `Fn+Shift+Delete` | 刪除記憶詞 | 先將游標移到該候選上再按。Rime 內建功能 |

### 切分移位（方向鍵）

| 功能 | 按鍵 |
|------|------|
| 左移一個注音 | `↑`、`Shift+←` |
| 右移一個注音 | `Shift+→` |
| 右移一整個字 | `→` |
| 左移一個字 | `←`（非選字狀態時） |
| 回到輸入列 | `Tab`（選字狀態時） |

> **Shift+F/B**（字級移動）和**方向鍵**（注音級移動）搭配使用，
> 可以快速定位到要修改的位置。

### 中英文與模式切換

| 功能 | 按鍵 |
|------|------|
| 切換中/英文 | `Shift+空白`（推薦）或 `Control+Shift+2` |
| 切換方案 | `Control+~` 或 `F4` |
| 切換全/半形 | `Control+Shift+3` |
| 切換繁/簡 | `Control+Shift+5` |
| 切換標點模式 | `Control+Shift+4` |

## 外掛語言與特殊功能

洋蔥 Plus 版內建多種外掛語言輸入，在注音模式下輸入特定前綴啟動：

| 功能 | 前綴 | 提示 | 說明 |
|------|:----:|------|------|
| 英文 | `3` | 《Easy》 | 英漢字典查詞 |
| 拉丁字母 | `4` | 《拉》 | 含重音符號的拉丁字母 |
| 日文 | `6` | 《日》 | 平假名、片假名、漢字 |
| 俄文 | `66` | 《俄》 | 西里爾字母 |
| 韓文 | `7` | 《韓》 | 韓語 HNC |
| 希臘文 | `77` | 《希》 | 希臘字母 |
| 全形 | `][` | 《全》 | 全形英數字 |
| 注音文 | `';` | 《注音文》 | 輸出注音符號本身 |
| emoji | `` ` `` | 《emoji集》 | 表情符號系列 |
| 倉頡反查 | `]` + 倉頡碼 | 《查》 | Dvorak `]` = QWERTY `=` 物理位置 |
| 數學/功能 | `'/` | — | 日期時間、計算機等 lua 功能 |

## 標點符號

Dvorak 版覆蓋了部分標點鍵，讓常用標點維持 QWERTY 的**物理位置**。

### 直接輸入

| 按鍵 | 輸出 | 說明 |
|:----:|:----:|------|
| `<` (Shift+,) | ， | 逗號 |
| `>` (Shift+.) | 。 | 句號 |
| `?` (Shift+/) | ？ | 問號 |
| `:` (Shift+;) | ： | 冒號 |
| `"` (Shift+') | ； | 分號 |
| `~` | ～ | 波浪號 |
| `!` | ！ | 驚嘆號 |
| `(` | （ | 左括號 |
| `)` | ） | 右括號 |
| `{` | 〔 | 左方括號 |
| `}` | 〕 | 右方括號 |

### 多選標點（Dvorak 專屬覆蓋）

| Dvorak 鍵 | 首選 | QWERTY 物理位置 | 說明 |
|:---------:|:----:|:---------------:|------|
| `-` | 、 | `'` (Enter 旁) | **頓號**系列 |
| `/` | 「 | `[` | **左引號**系列（含〈〔等） |
| `=` | 」 | `]` | **右引號**系列（含〉〕等） |
| `]` | ＝ | `=` | **等號**系列（含々〃等） |

### 其他標點（維持 punct_bopomo 原設定）

| 按鍵 | 首選 | 說明 |
|:----:|:----:|------|
| `'` | 、 | 頓號（僅非輸入狀態，因 `'` 同時是 ㄆ） |
| `[` | 「 | 左引號（僅非輸入狀態，因 `[` 同時是 ㄦ） |
| `\` | 刪除 | composing 時是注音字符刪除 / 非 composing 時是反斜線 |

## 與 QWERTY 原版的完整對照

### 選字

| 功能 | QWERTY 原版 | Dvorak 版 | 備註 |
|------|:-----------:|:---------:|------|
| 候選數 | 8 | 8 | |
| 右手選字 1-8 | Shift+Y/H/N/U/J/M/I/K | Shift+H/T/N/S/M/W/V/Z | Dvorak 用中排+下排 |
| 左手選字 1-8 | Shift+Q/A/W/S/X/E/D/C | — | **移除**，改為其他功能 |

### 編輯快捷鍵

| 功能 | QWERTY 原版 | Dvorak 版 | 備註 |
|------|:-----------:|:---------:|------|
| 補一聲 | `Shift+L` | — | **移除**，直接用空白鍵 |
| 上屏原始輸入 | `Shift+O` | `Shift+O` | **O**=Original |
| 上屏注音文 | `Shift+F` | `Shift+P` | **P**=Phonetic |
| 上屏 comment | `Shift+R` | — | **移除**，低頻功能 |
| BackSpace | `Shift+P` | — | **移除**，用 BackSpace 鍵 |
| Delete | `Shift+T` | — | **移除** |
| Escape | `Shift+G` | `Shift+Q` | **Q**=Quit |
| Home | `Shift+V` | `Shift+A` | Emacs C-**a** |
| End | `Shift+B` | `Shift+E` | Emacs C-**e** |
| Page Up | `Shift+Z` | `Shift+K` | vi **k**=up |
| Page Down | `Shift+空白` | `Shift+J` | vi **j**=down |
| 右移一個字 | — | `Shift+F` | Emacs C-**f**（Dvorak 新增） |
| 左移回退一個字 | — | `Shift+B` | Emacs C-**b**（Dvorak 新增） |

### 標點符號

| 功能 | QWERTY 按鍵 | Dvorak 按鍵 | 備註 |
|------|:-----------:|:-----------:|------|
| 頓號 | `'` | `-` | 同物理位置（Enter 旁） |
| 左引號「 | `[` | `/` | 同物理位置 |
| 右引號」 | `]` | `=` | 同物理位置 |
| 等號＝ | `=` | `]` | 同物理位置 |
| 逗號 | `<` | `<` | 不變 |
| 句號 | `>` | `>` | 不變 |

### 其他差異

| 項目 | QWERTY 原版 | Dvorak 版 |
|------|-------------|-----------|
| 注音鍵位 | 大千式 QWERTY | 大千式 Dvorak（同物理位置） |
| 選字標籤 | `𝟷𝚀𝚈` 等 (Unicode) | `H` `T` `N` 等 (ASCII) |
| 字典/prism | bopomo_onionplus | 字典共用，prism 獨立編譯 |
| 設定檔 | element_bopomo | element_bopomo_dvorak |
