# 洋蔥注音 Plus Dvorak 版 使用指南

本方案基於[洋蔥注音 Plus 版](https://github.com/oniondelta/Onion_Rime_Files)，
針對 **Dvorak 鍵盤 layout** 調整，讓使用者用 Dvorak 的物理鍵位輸入大千式注音。

## 安裝

將以下檔案放入 Rime 使用者目錄（macOS: `~/Library/Rime/`）：

- `bopomo_onionplus_dvorak.schema.yaml` — Dvorak 版方案定義
- `element_bopomo_dvorak.yaml` — Dvorak 版按鍵設定
- 其餘所有檔案與原版洋蔥注音 Plus 共用（字典、lua、opencc 等）

在 `default.custom.yaml` 中加入：

```yaml
patch:
  schema_list:
    - schema: bopomo_onionplus_dvorak
```

重新部署鼠鬚管即可使用。

---

## 設計原則

**物理鍵位不變**：所有 `Shift+字母` 快捷鍵維持與 QWERTY 原版相同的物理位置。
QWERTY 使用者切換到 Dvorak 版，手指位置和原版完全一致，不需重新記憶。

## 選字操作

候選字選單出現後，共 8 個候選字，使用右手選字。

### 右手選字

| 候選 | QWERTY 原版 | Dvorak 版 | 物理位置 |
|:----:|:-----------:|:---------:|:--------:|
| 1 | `Shift+Y` | `Shift+F` | 右手食指上排 |
| 2 | `Shift+H` | `Shift+D` | 右手食指中排 |
| 3 | `Shift+N` | `Shift+B` | 右手食指下排 |
| 4 | `Shift+U` | `Shift+G` | 右手中指上排 |
| 5 | `Shift+J` | `Shift+H` | 右手中指中排 |
| 6 | `Shift+M` | `Shift+M` | 右手中指下排 |
| 7 | `Shift+I` | `Shift+C` | 右手無名指上排 |
| 8 | `Shift+K` | `Shift+T` | 右手無名指中排 |

> QWERTY 原版另有左手選字（Shift+Q/A/W/S/X/E/D/C），Dvorak 版已移除，
> 改為 vi 風格翻頁鍵（見下方）。

### 翻頁

| 操作 | QWERTY 原版 | Dvorak 版 | 備註 |
|------|:-----------:|:---------:|------|
| 上一頁 | `Shift+Z` | `Shift+K` | vi 風格 k=up |
| 下一頁 | — | `Shift+J` | vi 風格 j=down |
| 下一頁 | `Shift+空白` | `Shift+空白` | 物理位置相同 |
| 上一頁 | `←` | `←` | 選字狀態時 |
| 下一頁 | `→` | `→` | 選字狀態時 |

## 快捷鍵

所有快捷鍵維持與 QWERTY 原版相同的**物理位置**：

### 編輯操作

| 功能 | QWERTY 原版 | Dvorak 版 | 物理位置 | 說明 |
|------|:-----------:|:---------:|:--------:|------|
| 補一聲 / 空格 | `Shift+L` | `Shift+N` | 右手無名指中排 | 選字時補一聲，或輸出空格 |
| 上屏原始輸入 | `Shift+O` | `Shift+R` | 右手無名指上排 | 直接上屏鍵盤原始字元 |
| 上屏注音文 | `Shift+F` | `Shift+U` | 左手食指中排 | 上屏 preedit 注音文字 |
| 上屏 comment | `Shift+R` | `Shift+P` | 左手食指上排 | 上屏提示碼內容 |
| 退格 | `Shift+P` | `Shift+L` | 右手小指上排 | 刪除前一字 |
| 刪除 | `Shift+T` | `Shift+Y` | 左手食指上排 | 刪除後一字 |
| 取消 | `Shift+G` | `Shift+I` | 左手食指中排 | Escape，清除輸入 |
| Home | `Shift+V` | `Shift+K` | 左手食指下排 | 移到開頭 |
| End | `Shift+B` | `Shift+X` | 左手中指下排 | 移到結尾 |

### 切分移位（QWERTY / Dvorak 相同）

| 功能 | 按鍵 |
|------|------|
| 左移一個注音 | `↑`、`Shift+←` |
| 右移一個注音 | `Shift+→` |
| 右移一整個字 | `→` |
| 回到輸入列 | `Tab`（選字狀態時） |

### 中英文與模式切換（QWERTY / Dvorak 相同）

| 功能 | 按鍵 |
|------|------|
| 切換中/英文 | `Control+Shift+2` 或 `Shift+空白`（非選字時） |
| 切換方案 | `Control+~` 或 `F4` |
| 切換全/半形 | `Control+Shift+3` |
| 切換繁/簡 | `Control+Shift+5` |
| 切換標點模式 | `Control+Shift+4` |

## 外掛語言與特殊功能（QWERTY / Dvorak 相同）

洋蔥 Plus 版內建多種外掛語言輸入，在注音模式下輸入特定前綴啟動：

| 功能 | 前綴 | 提示 | 說明 |
|------|:----:|------|------|
| 英文 | `3` | 《Easy》 | 英漢字典查詞 |
| 拉丁字母 | `4` | 《拉》 | 含重音符號的拉丁字母 |
| 日文 | `6` | 《日》 | 平假名、片假名、漢字 |
| 俄文 | `66` | 《俄》 | 西里爾字母 |
| 韓文 | `7` | 《韓》 | 韓語 HNC |
| 希臘文 | `77` | 《希》 | 希臘字母 |
| 全形 | `][` | 《全》 | 全形英數字 |
| 注音文 | `';` | 《注音文》 | 輸出注音符號本身 |
| emoji | `` ` `` | 《emoji集》 | 表情符號系列 |
| 倉頡反查 | `=` | 《查》 | 用倉頡碼反查注音 |
| 數學/功能 | `'/` | — | 日期時間、計算機等 lua 功能 |

> 前綴都在數字列或符號鍵上，QWERTY / Dvorak 物理位置相同。

### 英文模式範例

1. 按 `3` 進入英文模式（提示《Easy》）
2. 輸入英文單字
3. 選字上屏

### emoji 模式範例

1. 按 `` ` ``（ESC 下方的鍵）進入 emoji 模式
2. 輸入注音拼出分類名稱（如「笑」「愛」「食」）
3. 選擇想要的 emoji

## 標點符號

標點符號由 `punct_bopomo` 處理。Dvorak 版覆蓋了 `-` 鍵為頓號（對應 QWERTY `'` 的物理位置）。

### 直接輸入（半形模式）

| 按鍵 | 輸出 | QWERTY 按鍵 | 說明 |
|:----:|:----:|:-----------:|------|
| `<` (Shift+,) | ， | `<` (Shift+,) | 逗號（物理位置相同） |
| `>` (Shift+.) | 。 | `>` (Shift+.) | 句號（物理位置相同） |
| `?` (Shift+/) | ？ | `?` (Shift+/) | 問號（物理位置相同） |
| `:` (Shift+;) | ： | `:` (Shift+;) | 冒號（物理位置不同） |
| `"` (Shift+') | ； | `"` (Shift+') | 分號（物理位置不同） |
| `~` | ～ | `~` | 波浪號 |
| `!` | ！ | `!` | 驚嘆號 |
| `(` | （ | `(` | 左括號 |
| `)` | ） | `)` | 右括號 |
| `{` | 〔 | `{` | 左方括號 |
| `}` | 〕 | `}` | 右方括號 |

### 多選標點（按鍵後從候選選擇）

| 按鍵 | 首選 | QWERTY 按鍵 | 物理位置 | 說明 |
|:----:|:----:|:-----------:|:--------:|------|
| `-` | 、 | `'` | Enter 旁（相同物理位置） | **頓號**系列。Dvorak 專屬覆蓋 |
| `'` | 、 | `'` | Q（不同物理位置） | 頓號系列（僅非輸入狀態可用，因 `'` 同時是 ㄆ） |
| `[` | 「 | `[` | | 左引號系列（含〈〔等） |
| `]` | 」 | `]` | | 右引號系列（含〉〕等） |
| `=` | ＝ | `=` | | 等號系列（含々〃等） |
| `\` | ＼ | `\` | | 反斜線系列（composing 時作為音節分隔符） |
| `\|` | ｜ | `\|` | | 豎線系列 |

## 與 QWERTY 原版的差異摘要

| 項目 | QWERTY 原版 | Dvorak 版 |
|------|-------------|-----------|
| 注音鍵位 | 大千式 QWERTY | 大千式 Dvorak（同物理位置） |
| 選字鍵數 | 8 個右手 + 8 個左手 | 8 個右手（左手移除，改為 vi 翻頁） |
| 選字標籤 | `𝟷𝚀𝚈` 等 (Unicode 等寬) | `1F` `2D` `3B` 等 (ASCII) |
| 翻頁 | Shift+Z / Shift+空白 | Shift+K / Shift+J（vi 風格）/ Shift+空白 |
| 快捷鍵 | Shift+QWERTY 字母 | Shift+Dvorak 字母（同物理位置） |
| 頓號 | `'` 鍵 (Enter 旁) | `-` 鍵 (Enter 旁，同物理位置) |
| 字典/prism | bopomo_onionplus | 字典共用，prism 獨立編譯 |
| 設定檔 | element_bopomo | element_bopomo_dvorak |
